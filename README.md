CodSoft Python Internship

This repository contains the Python programming tasks completed during the CodSoft Internship.

Tasks

1. Calculator

A simple calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division based on user input.

2. Password Generator

A Python program that generates strong and random passwords according to the length specified by the user.

3. Rock Paper Scissors Game

A console-based game where the user plays Rock, Paper, Scissors against the computer, and the winner is decided based on the game rules.
